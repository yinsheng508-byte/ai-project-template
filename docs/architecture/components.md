# 组件注册表

> 实现任何 UI 元素前必读。已有组件必须复用，不得重新实现。
> 每次新增组件后立即更新本文件。

## 使用说明

1. 实现 UI 前先查本表，确认是否已有可复用的组件
2. 已有组件直接复用，不得重新实现一套样式或交互
3. 需要扩展已有组件时，在原组件上扩展，不新建替代品
4. 新增组件后立即登记到本表
