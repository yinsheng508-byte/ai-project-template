# INDEX

这是项目的索引文件。先看这里，再决定下一步读什么。

## 默认阅读顺序
1. `README.md`
2. `docs/INDEX.md`
3. `docs/workflow.md`
4. `docs/current/requirements.md`
5. `docs/current/tasks.md`

## 按阶段读取

### 需求澄清
- `docs/current/requirements.md`

### 任务编排
- `docs/current/requirements.md`
- `docs/current/tasks.md`
- `docs/templates/task-template.md`

### 执行开发
- `docs/current/tasks.md`
- `docs/current/review.md`（若任务有阻塞）

### Review / Escalation
- `docs/current/tasks.md`
- `docs/current/review.md`
- `docs/templates/review-checklist.md`

### 最终验收
- `docs/current/requirements.md`
- `docs/current/tasks.md`
- `docs/current/review.md`
- `docs/current/acceptance.md`
- `docs/templates/acceptance-checklist.md`

## 其他读取入口

### 查看历史轮次
- `docs/archive/`

### 查看模板
- `docs/templates/`

## 原则
- 不要一次读完整个仓库所有文档
- 先看索引，再按当前任务按需读取
- 当前阶段只读最相关文档，避免上下文污染
